AngularJS Controllers

HTML code:

  • <div ng-app="myApp" ng-controller="userCtrl"> <label>First Name:</label><input type="text" ng-model="user.fName" placeholder="Enter your first name."> </br> <label>Last Name:</label><input type="text" ng-model="user.lName" placeholder="Enter your last name."></br> <label>Full Name: </label><span class="">{{user.getFullName()}}</span> </div> <script type="text/javascript"> var app = angular.module('myApp', []) app.controller('userCtrl', ['$scope', function ($scope) { $scope.user = { fName: "", lName: "", getFullName: function() { var user; user = $scope.user; return user.lName + " " + user.fName; } }; }]) </script>

Output:



  • {{user.getFullName()}}
