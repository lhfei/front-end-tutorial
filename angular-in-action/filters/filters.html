AngularJS Filters

Html code:

  • <div ng-app="myApp" ng-controller="namesCtrl"> <table class="table table-hover" border="1" width="100%"> <thead> <tr> <th ng-click="orderByMe('name')">Name</th> <th ng-click="orderByMe('country')">Country</th> </tr> </thead> <tbody> <tr ng-repeat="x in names | orderBy:myOrderBy"> <td>{{x.name}}</td> <td>{{x.country}}</td> </tr> </tbody> </table> </div> <script type="text/javascript"> var app = angular.module('myApp', []); app.controller('namesCtrl', ($scope) => { $scope.names = [ {name:'Jani',country:'Norway'}, {name:'Carl',country:'Sweden'}, {name:'Margareth',country:'England'}, {name:'Hege',country:'Norway'}, {name:'Joe',country:'Denmark'}, {name:'Gustav',country:'Sweden'}, {name:'Birgit',country:'Denmark'}, {name:'Mary',country:'England'}, {name:'Kai',country:'Norway'} ]; $scope.orderByMe = (factor) => { $scope.myOrderBy = factor; } }); </script>

Output:

  • Click the table headers to change the sorting order:
    Name Country
    {{x.name}} {{x.country}}
